# Healthcare-EDA
Exploratory Data Analysis On a Synthetic Healthcare Patient  Records  Dataset | Python , Pandas , Matplotlib ,  Seaborn
Exploratory Data Analysis on a synthetically generated healthcare patient 
records dataset of 7,658 patients.

## Tools & Libraries
Python, Pandas, NumPy, Matplotlib, Seaborn, Plotly

## What's Covered
- Data cleaning & feature engineering
- Univariate, Bivariate & Multivariate Analysis
- 10+ visualizations (histograms, heatmaps, bubble chart, line chart)
- Severity scoring & age group segmentation

## Dataset
Synthetically generated using NumPy with a fixed random seed (reproducible).

## Note
Since the dataset is synthetic, variables are not correlated by design. 
Findings are for analytical practice and do not reflect real medical patterns.
